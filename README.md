# Introdução
 Projeto Final para a disciplina de Algoritmos e Programação do curso de Tecnólogo de Análise e Desenvolvimento de Sistemas (TADS), oferecido pela EAJ/UFRN.<p>

# Explicação Geral
O projeto é basicamente um Jogo de Quiz em que o usuário deve responder ao máximo de perguntas possíveis antes do tempo acabar.<p>
 Conténdo um sistema de Pontuações, onde, para cada acerto, é adicionado +10 pontos, e para cada erro, é subtraído -5 pontos;<p>
 E trabalha com conseitos estudados em Sala de Aula, como:
 - Conceitos Básicos (cin, cout, if, else, for, bool, etc)
 - Manipulação de Strings;
 - Manipulação de Vetores;
 - Funções próprias;
 - Dados Heterogênos (Structs);
 - Manipulção de Arquivos.
